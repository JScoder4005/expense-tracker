## [0.1.0] - 2025-12-27

### Added
- Initial release of Expense Tracker
- Dashboard with overview of expenses and income
- Category management with emoji icons and color coding
- Expense and income tracking
- Analytics with interactive charts
- Dark mode support with theme toggle
- 3D visualizations using Three.js
- Responsive design for all devices
- Type-safe forms with React Hook Form and Zod
- Real-time data synchronization with React Query
- Beautiful UI components from shadcn/ui
- Collapsible sidebar navigation

### Technical Stack
- React 19 with TypeScript
- Vite for build tooling
- Tailwind CSS v4 for styling
- React Query for state management
- Three.js for 3D effects
- Recharts for data visualization
